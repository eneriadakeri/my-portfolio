JavaScript, API, and JSON

This project was developed as part of the "Web Programming" course. The task involved creating a website using HTML, CSS, and JavaScript to retrieve and display data from a museum's API. I chose one of the available museum APIs, such as the Cleveland Museum of Art, and used the provided data to showcase key information, like museum departments, exhibitions, or types of artwork. The project required making GET HTTP requests to specific API endpoints, handling JSON responses, and dynamically displaying the results on the webpage.
